# 芙莉莲独立角色设计稿 v0.5 机制重构稿

目标游戏：杀戮尖塔2  
目标角色：芙莉莲  
本稿约束：不改卡牌名称，不改卡牌数量，不改遗物名称，不改遗物数量，不改药水名称，不改药水数量。只重写关键词、规则、卡牌效果、遗物效果、药水效果。  
本稿定位：让芙莉莲成为能融入原游戏强度、节奏和构筑逻辑的完整 mod 角色，而不是只靠原作皮肤和少数大招撑体验。

## 1. 结论

旧版 v0.4 的方向成立，但厚度不足。

主要问题有五个：

1. `解析` 只有线性追加伤害，缺少像 Doom、Focus、Star Energy 那样的阶段质变。
2. `隐匿魔力 -> 解放` 有爆发，但缺少中间玩法，玩家在未爆发回合容易只是攒数字。
3. `回忆` 原著味强，但战斗反馈偏散，玩家不容易感到“成型了”。
4. `保留与长咏唱` 更像稳定工具，不像独立流派，延迟收益不够高。
5. 原游戏角色成型后常见的“高乘算爆发”和“困难无限”没有被主动设计，反而有若干低质量无限风险。

v0.5 的核心修正：

1. 新增 `看破`，把解析从线性增伤改成有爆发阶梯的负面状态。
2. 新增 `抑制`，把芙莉莲隐藏魔力的原著设定变成持续构筑轴。
3. 新增 `咏唱`，把延迟牌做成高倍率终端。
4. 新增 `回路`，提供类似原游戏无限构筑的成型骨架，但必须通过多组件组合实现。
5. 重写 `回忆` 的阶段奖励，让普通魔法路线有明确成长反馈。

## 2. 原游戏标尺

本地资料显示，杀戮尖塔2 单角色卡池通常约 87-89 张。常见结构为：起始牌约 4 张，普通约 20 张，罕见约 35-40 张，稀有约 25-30 张，古代 2 张。

基础数值标尺：

| 类型 | 原游戏常见价值 |
|---|---:|
| 1 费基础攻击 | 6 伤害 |
| 高质量 1 费普通攻击 | 7-10 伤害加小效果 |
| 高质量 1 费普通防御 | 7-9 格挡加小效果 |
| 2 费攻击 | 15-20 伤害或 10-13 群体 |
| 3 费攻击 | 24-32 伤害，或带明显返还条件 |
| 0 费抽牌 | 必须有消耗、弃牌、状态污染、回合限制或组合门槛 |
| 单回合无限 | 可以存在，但通常需要压缩牌组、多个组件、回能、抽牌和低费循环 |

原角色流派密度对照：

| 角色 | 主资源/主轴 | 成型爽点 |
|---|---|---|
| 铁甲战士 | 消耗、自伤、力量、易伤、格挡 | 消耗链、力量爆发、壁垒肚皮、攻击连打 |
| 猎手 | 毒、刀、弃牌、Sly、技能链 | 刀爆发、毒滚雪球、弃牌循环、弱化刺杀 |
| 故障机器人 | 充能球、集中、能力、0费回收、状态转化 | 闪电链、冰霜壁、黑球斩杀、能力机、All for One 循环 |
| 死灵绑定者 | Osty、Doom、Soul、Ethereal | 召唤成长、Doom 处决、Soul 引擎、幽灵爆发 |
| 储君 | 星能量、锻造、无色、造牌、技能链 | 星能量爆发、王刃成长、造牌转收益、顶牌控制 |

芙莉莲要合格，至少需要：

1. 一条稳定中速线。
2. 一条爆发高上限线。
3. 一条防御成长线。
4. 一条资源循环线。
5. 一条可困难成型的无限线。
6. 若干乘算组合，但不能两三张普通牌就达成。

## 4. 关键词规则

### 法术

内部标签。芙莉莲的大部分专属攻击、技能、能力都带 `法术`。普通 `打击`、`防御` 不算。

### 解析 X

敌方负面状态。每当你的 `法术` 对该敌人造成未被格挡的伤害时，消耗 1 层 `解析`，造成 2 点生命流失。

规则细节：

1. 多段法术每一段都可以各消耗 1 层解析。
2. 生命流失不受力量、易伤、虚弱影响。
3. 若一张 10 段法术命中有 10 层解析的敌人，会消耗 10 层解析并造成 20 点额外生命流失。
4. 解析不会自行造成伤害，必须由法术命中兑现。

设计意图：解析是芙莉莲的基础增伤资源，类似毒和易伤之间的状态。它需要铺垫，但多段牌能把铺垫转为爆发。

### 看破 X

敌方负面状态。你的下一张 `法术` 对该敌人消耗解析时，每消耗 1 层解析造成的生命流失从 2 提高到 3。若该法术至少消耗了 1 层解析，移除 1 层看破。

规则细节：

1. 每张法术最多消耗 1 层看破。
2. 看破只强化解析生命流失，不提高法术本体伤害。
3. 多层看破需要多张法术逐步消耗。
4. 若目标没有解析，看破不会被消耗。

设计意图：看破是解析流的阶梯收益。没有看破时是稳定伤害，有看破时进入杀人魔法爆发窗口。

### 隐匿魔力

玩家战斗内资源，默认上限 19。每当你获得隐匿魔力后，若隐匿魔力达到 10 且你的 `解放` 少于 2 层，失去 10 隐匿魔力并获得 1 层解放。

规则细节：

1. `解放` 上限为 2。
2. 若解放已满，隐匿魔力可以保留到 19，但不继续转化。
3. 当你消耗解放后，若隐匿魔力仍至少为 10，下一次获得隐匿魔力时重新检查转化。
4. 隐匿魔力不等于能量，不能直接支付费用。

### 解放 X

玩家增益。你的下一张 `法术` 额外打出 1 次，然后失去 1 层解放。

规则细节：

1. 由解放产生的额外打出不能再次触发解放。
2. 解放复制可以触发解析、看破和普通“打出法术”收益。
3. 一张牌最多消耗 1 层解放。
4. 外部复制和解放只做加法，不做递归。

设计意图：解放是芙莉莲的核心爆发开关。它可以强，但必须通过隐匿魔力或稀有牌铺垫。

### 抑制 X

玩家增益，默认上限 5。回合结束时，若你本回合没有打出攻击牌，获得 X 层隐匿魔力。

规则细节：

1. 打出非攻击法术不会阻止抑制。
2. 自动打出的攻击牌会阻止抑制。
3. 抑制是战斗内持续增益。
4. 抑制上限为 5，超过部分无效。

设计意图：把“隐藏魔力、不急于暴露实力”做成一个能防守、能蓄势的构筑轴。

### 咏唱 X

带有 `咏唱 X` 的效果在打出时支付费用并指定目标，然后进入咏唱区。你的回合开始时，咏唱倒计时减少 1；降到 0 时自动结算。

规则细节：

1. 咏唱结算算作你的法术结算，可以触发解析、看破和解放。
2. 咏唱结算不推进回路计数。
3. 若原目标死亡，随机选择合法敌人；若无合法目标，攻击部分失效，非攻击部分照常。
4. 咏唱牌因为延迟，应比同费即时牌高 80%-120% 终值。

数值标尺：

| 咏唱牌类型 | 合理终值 |
|---|---:|
| 1费咏唱1攻击 | 16-20 单体，或 12-15 加解析/看破 |
| 2费咏唱1攻击 | 30-38 单体，或 20-26 群体 |
| 3费咏唱1攻击 | 48-60 单体，或 30-40 群体 |
| 咏唱2 | 只用于罕见/稀有终端，比咏唱1再高 30%-50%，但应给少量即时补偿 |

### 回路

回合内计数。每当你手动打出第 3 张 `法术`，触发 1 次回路，然后重新计数。

规则细节：

1. 回路只统计手动打出的法术。
2. 解放额外打出的复制法术不计入回路计数。
3. 咏唱自动结算不计入回路计数。
4. 回路本身没有奖励，奖励来自卡牌、能力和遗物。

设计意图：回路是芙莉莲的困难无限骨架。它提供原游戏成型后“转起来”的空间，但需要抽牌、能量、低费法术和牌组压缩共同满足。

### 普通魔法

卡牌标签。通常是低数值、低费用、带功能的小法术。它们是回忆路线和回路路线的润滑剂。

### 回忆

战斗内计数资源。每当你本场战斗第一次手动打出一张不同名称的 `普通魔法`，获得 1 层回忆。

每获得第 3、6、9、12 层回忆时，按顺序循环触发旅途奖励：

| 里程碑序号 | 奖励 |
|---|---|
| 第 1 次里程碑 | 抽 1 张牌 |
| 第 2 次里程碑 | 获得 7 格挡 |
| 第 3 次里程碑 | 获得 3 隐匿魔力 |
| 第 4 次里程碑起 | 重复上述顺序 |

限制：

1. 由随机生成得到的普通魔法每回合最多触发 1 次回忆。
2. 复制打出的普通魔法不触发回忆。
3. 回忆不在战斗外保留。

## 5. 主要流派

| 流派 | 核心牌 | 爽点 | 成型难度 |
|---|---|---|---|
| 解析看破流 | 杀人魔法、弱点解析、战斗解析、解析一切、千年齐射 | 叠解析，给看破，用多段法术一次性兑现 | 中 |
| 隐匿解放流 | 魔力抑制、深度魔力抑制、静默蓄势、巨大魔力、全力杀人魔法 | 攒到解放后复制关键法术 | 中高 |
| 抑制防守流 | 安静脚步、魔力隐蔽、千年的从容、屏障反转、古代屏障 | 少攻击回合换隐匿魔力和防守成长 | 中 |
| 咏唱终端流 | 缓慢咏唱、长咏唱、咏唱保存、最后的杀人魔法 | 延迟高收益，安排下回合爆发 | 中高 |
| 普通魔法回忆流 | 清洁魔法、飞行魔法、魔法收藏、普通魔法精通、千年研究 | 小魔法越多，抽牌、防御、隐匿魔力越顺 | 中 |
| 回路无限流 | 魔力丝线、飞行魔法、师徒节奏、魔法百科全书、普通魔法精通 | 三张法术一轮，抽牌与能量闭环 | 高 |

困难无限的合格门槛：

1. 至少需要 `师徒节奏` 和 `魔法百科全书` 两个引擎组件。
2. 需要三张不同名称的低费法术反复进入手牌。
3. 需要牌组压缩或足够检索。
4. 需要费用降低、0费普通魔法或能量返还。
5. `中性无限` 可以由 1 张 1费抽牌法术加 2 张 0费法术达成，循环本身不增长能量。
6. `正收益无限` 必须让本次回路的三张法术打出时费用合计为 0，才允许同时增长抽牌和能量。
7. 不允许由单张 0费抽牌或普通随机生成直接形成无限。

## 11. 关键组合与风险

### 11.1 合格高乘算组合

组合一：解析爆发  
`弱点解析` 施加解析和看破，`千年齐射` 多段消耗解析，`解放` 复制千年齐射。  
结果：解析生命流失、多段攻击、复制共同放大。  
门槛：需要解析层数、看破、3费攻击和解放，不能早期稳定达成。

组合二：隐匿魔力爆发  
`巨大魔力` 或 `静默蓄势` 提供隐匿魔力，转化为解放，再复制 `全力杀人魔法` 或 `解析一切`。  
结果：高爆发或高铺垫。  
门槛：慢启动，怕抽牌不齐。

组合三：回忆资源链  
`普通魔法精通`、`魔导书收藏家`、`芙兰梅的手记` 共同强化普通魔法。  
结果：普通魔法提供抽牌、防御、隐匿魔力和回忆里程碑。  
门槛：需要多张不同普通魔法，不靠复制同一张牌。

组合四：回路困难无限  
`师徒节奏` 提供回路抽牌，`魔法百科全书` 提供回路回能，牌组压缩后用三张不同名称低费法术循环。  
中性循环样例：`魔力针`、`近距施法`、`魔力爆散`，在两张能力都在场时，三张牌费用合计为 1，`魔力针` 抽 1，两个能力共抽 2，`魔法百科全书` 回 1 能量，循环可以不断造成伤害但不增长能量。  
正收益循环样例：`近距施法`、`魔力爆散`、`咏唱保存+`，在两张能力都在场时，三张牌费用合计为 0，`师徒节奏` 额外抽 1，两个能力合计抽 3，`魔法百科全书` 回 1 能量，循环每轮净赚 1 能量。  
结果：存在原游戏式单回合无限运转，并且能进一步形成无限能量。  
门槛：至少两个能力组件、三张名称不同的低费法术、关键升级、牌组压缩或强检索。不是普通牌两三张直接无限。

### 11.2 明确禁止的低质量无限

1. 0费非消耗抽2不能存在。
2. 随机普通魔法生成不能无限触发回忆。
3. 解放复制不能触发解放。
4. 咏唱自动结算不能推进回路。
5. 外部复制不能递归复制芙莉莲复制。

### 11.3 需要实测的峰值

| 风险 | 触发方式 | 第一版控制 |
|---|---|---|
| 解析多段过强 | 解析一切加千年齐射加解放 | 看破只消耗 1 层，解析基础为 2 |
| 回路无限过早 | 师徒节奏加魔法百科全书加多张 0费法术 | 回路要求 3 张手动法术，百科全书要求 3 个不同名称才回能 |
| 普通魔法抽牌链 | 魔导书收藏家加普通魔法精通加魔法收藏 | 生成牌每回合最多触发 1 次回忆 |
| 解放常驻 | 巨大魔力、静默蓄势、储备连锁 | 解放上限 2，隐匿魔力上限 19 |
| 咏唱收益过弱 | 延迟牌没有即时收益 | 咏唱终值按同费即时牌 1.8-2.2 倍设计 |

## 12. 完成度判断

v0.5 相比 v0.4 的改进：

| 维度 | v0.4 状态 | v0.5 状态 |
|---|---|---|
| 丰富度 | 三条主线加两个副线，交叉不足 | 六个系统，主线和副线可互相借件 |
| 数值合理性 | 解析 3 点偏线性，咏唱偏弱，部分 0费抽牌危险 | 解析基础降为 2，看破抬到 3，咏唱按延迟高倍率补偿 |
| 玩家预期 | 芙莉莲味道有，但部分回合只是攒资源 | 抑制、咏唱、看破让“冷静观察后爆发”可见 |
| 成型爽点 | 爆发有，循环不足 | 爆发、回忆成长、回路无限均存在 |
| 成型难度 | 稀有牌影响过大，部分无限风险意外出现 | 无限需要多组件，爆发需要解析、看破、解放或咏唱 |
| 原游戏融入度 | 接近 Defect/Necrobinder，但资源链偏短 | 更接近原角色的多轴、乘算、难无限结构 |

结论：v0.5 达到“可进入实现与测试”的设计成熟度。它保留芙莉莲的原著识别点，同时补足原游戏角色需要的构筑厚度、成型爽点和困难无限空间。

## 13. 实现备注

1. `解析`、`看破` 是敌方状态。
2. `隐匿魔力`、`解放`、`抑制`、`回忆` 是玩家战斗状态。
3. `回路` 是回合内临时计数，回合开始清零。
4. `咏唱区` 需要记录原牌、剩余回合、原目标、是否由玩家手动打出。
5. 所有随机生成普通魔法的效果必须调用同一个小魔法池。
6. 所有复制效果必须检查“由解放产生的复制牌不再次触发解放”。
