# 第二组难度 7 最终：双目标同步斩杀

## 玩家题面

> 作废：第二组题目不再作为有效产出或框架依据。

从 15 张候选牌中选择 7 张作为起始牌组，从 3 瓶候选药水中选择 1 瓶，并从 2 个候选遗物中选择 1 个，在第 5 回合结束前同时击败本体与影子。

本题采用固定自动结算顺序。每回合抽牌后，系统按“同步标记 -> 双切 -> 校准击 -> 本体切 -> 影子切 -> 贪单斩 -> 镜像收束 -> 护扫 -> 打击 -> 防御”的顺序，依次打出手牌中能支付费用的牌；玩家只负责选择牌组、药水、遗物，不手动调整出牌顺序。

## 本题自定义规则

以下规则均为本题自定义机制，不是原版官方关键词；原版通用概念沿用《杀戮尖塔》官中称呼。

| 术语 | 规则 |
|---|---|
| 同步斩杀 | 本体与影子必须在同一回合结束前同时死亡 |
| 偏杀失败 | 若任一回合结束时只有一个目标死亡，立即判定失败 |
| 同步标记 | 跨回合保留，最高 6 层 |
| 镜像收束 | 对两个目标造成同步标记加成伤害，然后清空同步标记 |
| 失衡反噬 | 若回合结束时两个目标生命差距达到 18 或以上，玩家额外失去 4 点生命 |
