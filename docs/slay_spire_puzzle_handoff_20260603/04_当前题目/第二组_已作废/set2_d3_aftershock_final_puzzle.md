# 第二组难度 3 最终：余震与重击窗口

## 玩家题面

> 作废：第二组题目不再作为有效产出或框架依据。

从 15 张候选牌中选择 7 张作为起始牌组，在第 4 回合结束前击败敌人。

本题采用固定自动结算顺序。每回合抽牌后，系统按“脉冲 -> 震击 -> 稳震格挡 -> 早爆 -> 余震爆发 -> 破震斩 -> 快斩 -> 护步 -> 重斩 -> 打击 -> 防御”的顺序，依次打出手牌中能支付费用的牌；玩家只负责选择牌组，不手动调整出牌顺序。

## 本题自定义规则

以下规则均为本题自定义机制，不是原版官方关键词；原版通用概念沿用《杀戮尖塔》官中称呼。

| 术语 | 规则 |
|---|---|
| 余震 | 跨回合保留，最高 7 层 |
| 余震爆发 | 根据当前余震层数造成额外伤害，然后清空余震 |
| 破震 | 消耗 2 层余震换取补刀伤害 |
| 早爆 | 清空余震，并根据清空前的余震层数反噬玩家 |
| 过载反噬 | 回合结束时若仍有 6 层或以上余震，玩家额外失去 5 点生命 |

## 资源审计

| 维度 | 结论 |
|---|---|
| 第一层资源选择 | 必须同时选择余震来源、余震兑现、破震补刀或格挡承压 |
| 只堆输出 | 重斩诱导线最高 43.0476%，低于主线且不能稳定通关 |
| 早爆陷阱 | 早爆最高 1.8095%，基本不能靠好牌序逃课 |
| 排序作用 | 固定顺序只影响余震何时兑现；缺少余震来源或爆发牌无法进入主线 |
