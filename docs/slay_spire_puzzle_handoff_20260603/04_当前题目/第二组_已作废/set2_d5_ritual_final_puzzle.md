# 第二组难度 5 最终：仪式打断与献祭层数

## 玩家题面

> 作废：第二组题目不再作为有效产出或框架依据。

从 14 张候选牌中选择 6 张作为起始牌组，从 3 瓶候选药水中选择 1 瓶，并从 2 个候选遗物中选择 1 个，在第 5 回合结束前击败敌人。

本题采用固定自动结算顺序。每回合抽牌后，系统按“献祭刻痕 -> 仪式格挡 -> 断仪斩 -> 伪打断 -> 封仪切 -> 献祭收束 -> 稳击 -> 重斩 -> 打击 -> 防御”的顺序，依次打出手牌中能支付费用的牌；玩家只负责选择牌组、药水、遗物，不手动调整出牌顺序。

## 本题自定义规则

以下规则均为本题自定义机制，不是原版官方关键词；原版通用概念沿用《杀戮尖塔》官中称呼。

| 术语 | 规则 |
|---|---|
| 仪式进度 | 每回合开始时增加 1；到达 5 时本题失败 |
| 献祭层数 | 跨回合保留，最高 8 点 |
| 献祭收束 | 按当前献祭层数造成额外伤害，然后清空献祭层数 |
| 伪打断 | 看似补伤害，但会推进仪式进度并反噬玩家 |
| 洗牌 | 抽牌堆不足时，立即洗入弃牌堆继续抽 |
