# 第二组难度 10 最终：双重倒计时与资源拍卖

## 玩家题面

> 作废：第二组题目不再作为有效产出或框架依据。

从 15 张候选牌中选择 7 张作为起始牌组，从 3 瓶候选药水中选择 1 瓶，并从 2 个候选遗物中选择 1 个，在第 5 回合结束前击败敌人。

本题采用固定自动结算顺序。每回合抽牌后，系统按“左叫价 -> 右叫价 -> 双叫价 -> 压价 -> 拍卖护栏 -> 成交拍品 -> 稳击 -> 重斩 -> 打击 -> 防御 -> 空牌”的顺序，依次打出手牌中能支付费用的牌；玩家只负责选择牌组、药水、遗物，不手动调整出牌顺序。

## 本题自定义规则

以下规则均为本题自定义机制，不是原版官方关键词；原版通用概念沿用《杀戮尖塔》官中称呼。

| 术语 | 规则 |
|---|---|
| 左倒计时/右倒计时 | 每回合开始两边各 +1；任一边达到 5，立即失败 |
| 竞价值 | 跨回合保留，最高 8 点 |
| 叫价牌 | 降低对应倒计时，并增加竞价值 |
| 成交拍品 | 若两边倒计时都不高于 2，按竞价值造成额外伤害；否则只造成低伤害 |
| 负债反噬 | 回合结束时，两边倒计时合计过高会造成额外生命损失 |
