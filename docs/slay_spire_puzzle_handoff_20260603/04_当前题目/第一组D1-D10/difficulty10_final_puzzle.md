# 难度 10 最终题目：时间裂隙与三相斩杀

> 2026-06-06 真实性复核：本题含 `裂隙抽取`、`相位门槛`、`时间药水`、`怀表核心`、`棱镜碎片` 等未核验/疑似自造内容，并使用自动优先级。作为普通尖塔残局必须重做；不要把这些词润色成原游戏文案。

## 玩家题面

从 46 张候选牌中按限制选择 16 张作为裂隙牌组，从 3 种药水中选择 1 种，并从 3 个遗物中选择 1 个带入战斗，在第 4 回合结束前击败敌人。

本题只按题面写明的效果结算。所有改动过的原版牌都用“（改）”标记。

### 本题关键词

| 关键词 | 规则 |
|---|---|
| 裂隙抽取 | 每回合从完整裂隙牌组中随机抽 2 张；回合结束后裂隙重置，下回合仍从完整裂隙牌组抽取 |
| 回路优先级 | 回合开始先结算延迟伤害，再结算遗物；之后可以选择是否使用药水；最后按候选牌池表格顺序自动打出至多 2 张当前能支付费用的手牌 |
| 不能跳过 | 回路优先级中不能跳过更靠前且能支付费用的牌，也不能提前停手 |
| 相位门槛 | 时间药水允许第 1 回合斩杀，回声药水允许第 2 回合及以后斩杀，碎甲药水允许第 3 回合及以后斩杀；早于相位门槛的致命伤害会把敌人压到 1 点生命 |
| 充能 | 作为部分攻击和格挡的数值资源，跨回合保留 |
| 标记 | 作为部分攻击、终局指令和延迟爆破的数值资源，跨回合保留；第 3 回合结束后失去 2 层标记 |
| 回声 | 下一次攻击伤害翻倍；多层回声可连续作用于多次攻击 |
| 延迟伤害 | 延迟爆破（改）加入的延迟伤害会在下回合开始时结算 |
| 过热 | 每层过热使敌人攻击额外造成 4 点伤害；过热跨回合保留，除非被排热或冷却回路降低 |
| 敌人护甲 | 敌人护甲会保留；攻击和延迟伤害先扣护甲，再扣生命 |

### 判定

| 维度 | 结论 |
|---|---|
| 难度目标 | 10/10 |
| 卡池规模 | 46 张候选牌，16 张裂隙牌组，叠加药水、遗物、相位门槛、延迟伤害、过热和标记 |
| 核心机制 | 裂隙抽取、自动优先级、三相斩杀门槛、跨回合延迟爆破、过热反噬、敌人保留护甲 |
| 三档回合 | 存在第 1、第 2、第 3 回合首次击杀线，三条代表路线依赖不同药水相位 |
| 高风险高收益 | 时间快线总成功率最高但失败率仍有 45.8742%；碎甲三相线失败率超过 90% |
| 偷跑控制 | 全量 3033 个合法构筑中无 100% 稳定线；最高线属于时间相位快线，没有绕开核心机制 |
