# 难度 4 重做草案：灼伤洗牌与消耗窗口

## 玩家题面

从 18 张候选牌中选择 9 张作为起始牌组，从 3 种药水中选择 1 种带入战斗，在第 5 回合结束前击败敌人。

本题按正常《杀戮尖塔》战斗方式结算：玩家在战斗内自由选择出牌顺序、药水使用时机和目标。所有改动过的原版牌都用“（改）”标记；未标记“（改）”的牌按原版效果理解。

## 初始状态

| 项目 | 数值 |
|---|---:|
| 玩家生命 | 38 |
| 每回合能量 | 3 |
| 每回合抽牌 | 5 |
| 起始牌组 | 从候选牌池中选择 9 张 |
| 药水 | 从药水池中选择 1 种 |
| 遗物 | 无 |
| 敌人生命 | 112 |
| 回合上限 | 第 5 回合结束前击杀 |

## 敌人行动

敌人护甲不会保留。灼伤只会进入玩家牌堆，敌人没有抽牌堆、弃牌堆或手牌。

| 回合 | 敌人行动 |
|---:|---|
| 1 | 攻击 8；回合结束将 1 张灼伤加入玩家弃牌堆 |
| 2 | 攻击 14；回合结束将 1 张灼伤加入玩家弃牌堆 |
| 3 | 攻击 20；回合结束将 2 张灼伤加入玩家弃牌堆 |
| 4 | 攻击 28 |
| 5 | 攻击 38；若敌人未死亡则玩家失败 |

## 药水池

| 药水 | 效果 |
|---|---|
| 火焰药水 | 一场战斗限用 1 次，造成 20 点固定伤害；不受易伤加成 |
| 能量药水 | 一场战斗限用 1 次，本回合获得 2 点能量 |
| 幽灵药水 | 一场战斗限用 1 次，本回合获得 1 层无实体 |

## 状态规则

| 状态 | 规则 |
|---|---|
| 灼伤 | 不可打出；回合结束时若在手牌中，玩家失去 2 点生命 |
| 伤口 | 不可打出；没有额外效果 |
| 晕眩 | 不可打出；消耗 |
| 易伤 | 攻击伤害变为 1.5 倍，向下取整 |
| 虚弱 | 敌人本回合攻击伤害变为 0.75 倍，向下取整 |
| 消耗 | 被消耗的牌本场战斗不再进入抽牌堆或弃牌堆 |
| 抽牌不足 | 抽牌堆不足时，立刻将玩家弃牌堆洗入抽牌堆继续抽 |

## 代表解法方向

| 解法方向 | 资源包 | 定位 |
|---|---|---|
| 火焰吐息快线 | 火焰吐息、进化、低费攻击、火焰药水 | 利用灼伤进入牌堆后的抽牌触发补伤害，争取第 3-4 回合击杀 |
| 消耗防守主线 | 无惧疼痛、燃烧契约、重振精神、耸肩无视、幽灵药水 | 把灼伤和防御转化为格挡窗口，承受第 4-5 回合压力后击杀 |
| 易伤爆发线 | 重击或上勾拳、剑柄打击、顺劈斩、火焰药水或能量药水 | 不依赖状态收益，靠易伤和能量窗口抢杀，但容易被后续灼伤拖死 |
