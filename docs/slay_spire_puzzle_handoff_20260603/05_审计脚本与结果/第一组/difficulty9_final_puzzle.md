# 难度 9 最终题目：破镜神格与护甲反射

## 玩家题面

从 28 张候选牌中按限制选择 13 张作为镜像牌组，从 3 种药水中选择 1 种，并从 3 个遗物中选择 1 个带入战斗，在第 3 回合结束前击败敌人。

本题只按题面写明的效果结算。所有改动过的原版牌都用“（改）”标记。

### 选牌限制

| 限制 | 内容 |
|---|---|
| 必选 | 虔信（改）x1、五体投地（改）x2、敬拜（改）x2、诸神黄昏（改）x1、审判（改）x1 |
| 真言底座 | 祈祷（改）至少 1 张，光辉（改）至少 1 张，圣洁（改）至少 1 张 |
| 真言组件 | 虔信（改）、五体投地（改）、祈祷（改）、敬拜（改）合计至少 5 张 |
| 攻击组件 | 光辉（改）、诸神黄昏（改）、审判（改）、刻现实（改）、惩击、供奉（改）、打击合计至少 5 张 |
| 防守组件 | 五体投地（改）、祈祷（改）、圣洁（改）、护墙、防御、化体（改）合计至少 4 张 |
| 总牌数 | 最终镜像牌组为 13 张 |

### 本题关键词

| 关键词 | 规则 |
|---|---|
| 镜像抽取 | 每回合从镜像牌组中随机抽 3 张；回合结束后镜像重置，下回合仍从完整镜像牌组抽取 |
| 回路优先级 | 每回合抽牌后，先结算回合开始遗物，再可以选择是否使用药水，之后按候选牌池表格顺序自动打出至多 2 张当前能支付费用的手牌 |
| 不能跳过 | 回路优先级中不能跳过更靠前且能支付费用的牌，也不能提前停手 |
| 真言 | 真言达到 10 时立刻失去 10 点真言并进入神格，本回合攻击伤害变为 3 倍；回合结束离开神格 |
| 护甲反射 | 你在非神格下打出攻击牌时，若该次攻击的基础结算伤害至少为 12，敌人先获得 10 点护甲，再结算该次伤害 |
| 敌人护甲 | 敌人护甲会保留；攻击牌先扣护甲，再扣生命 |
| 消耗 | 本题为镜像牌组，消耗只影响当前回合手牌，不会使该牌从后续回合的镜像抽取中永久移除 |

### 判定

| 维度 | 结论 |
|---|---|
| 难度目标 | 9/10 |
| 卡池规模 | 28 张候选牌，13 张镜像牌组，叠加药水、遗物、真言、神格、护甲反射 |
| 核心机制 | 镜像抽取、自动优先级、真言阈值、神格爆发、护甲反射、破镜降甲 |
| 三档回合 | 存在第 1、第 2、第 3 回合首次击杀线，且概率梯度明显 |
| 高风险高收益 | 最快线总成功率 36.5650%，失败率 63.4350%；三回合线成功率仅 2.2935% |
| 偷跑控制 | 全量 873 个合法构筑中无 100% 稳定线；最高线未超过 40% |

## 资源审计

| 维度 | 结论 |
|---|---|
| 第一层资源选择 | 必须选对真言来源、神格药水/破镜药水、日晷/经文/达玛鲁等资源包 |
| 错误药水 | 破镜达玛鲁三回合线最高 2.2935%，不能靠好牌序稳定过关 |
| 半正确资源 | 真言达玛鲁蓄力线最高 18.7686%，低于神格药水资源包 |
| 排序作用 | 镜像抽取和真言达标时点决定爆发窗口，但成败首先由神格资源包选择决定 |
