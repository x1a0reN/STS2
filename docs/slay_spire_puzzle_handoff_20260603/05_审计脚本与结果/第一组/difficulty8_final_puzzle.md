# 难度 8 最终题目：绝缘破裂与黑球过载

## 玩家题面

从 24 张候选牌中按限制选择 12 张作为起始牌组，从 3 种药水中选择 1 种，并从 3 个遗物中选择 1 个带入战斗，在第 5 回合结束前击败敌人。

本题只按题面写明的效果结算。所有改动过的原版牌都用“（改）”标记。

### 选牌限制

| 限制 | 内容 |
|---|---|
| 必选 | 双重释放（改）x1、递归（改）x1、循环（改）x1、寒意（改）x1 |
| 充能球底座 | 电击（改）至少 1 张，黑暗（改）至少 1 张 |
| 充能球组件 | 电击（改）、黑暗（改）、寒意（改）、寒流（改）、球状闪电（改）、冷静头脑（改）合计至少 6 张 |
| 防守组件 | 寒意（改）、寒流（改）、防御、飞跃、冷静头脑（改）合计至少 4 张 |
| 输出组件 | 寒流（改）、球状闪电（改）、熔化（改）、精简改良（改）、打击合计至少 3 张 |
| 总牌数 | 最终起始牌组为 12 张 |

| 项目 | 数值 |
|---|---:|
| 玩家生命 | 45 |
| 每回合能量 | 3 |
| 每回合抽牌 | 5 |

| 药水 | 从药水池中选 1 种 |
| 遗物 | 从遗物池中选 1 个 |
| 敌人生命 | 60 |

### 本题关键词

| 关键词 | 规则 |
|---|---|
| 充能球槽 | 最多容纳 3 个充能球；充能球满时再生成充能球，会先释放最左侧充能球 |
| 回路优先级 | 每回合开始时可以选择是否使用药水；之后按候选牌池表格顺序自动打出至多 2 张当前能支付费用的手牌 |
| 不能跳过 | 回路优先级中不能跳过更靠前且能支付费用的牌，也不能提前停手 |
| 闪电 | 回合结束被动造成 3+集中伤害；释放时造成 8+集中伤害 |
| 冰霜 | 回合结束被动获得 2+集中格挡；释放时获得 7+集中格挡 |
| 黑暗 | 回合结束被动使自身储值增加 6+集中；释放时造成等同储值的伤害 |
| 绝缘破裂 | 每当你通过冰霜充能球获得格挡，敌人失去等量生命；该生命流失不受护甲阻挡 |
| 循环 | 循环（改）生效后，回合结束时最左侧充能球额外触发 1 次被动效果 |

### 敌人行动

敌人护甲会保留；攻击牌和充能球伤害先扣护甲，再扣生命；绝缘破裂直接扣生命。

| 回合 | 敌人行动 |
|---:|---|
| 1 | 攻击 10 |
| 2 | 攻击 18；回合结束获得 18 点保留护甲 |
| 3 | 攻击 26 |
| 4 | 攻击 34；回合结束获得 26 点保留护甲 |
| 5 | 处决 45 |

### 药水池

| 药水 | 效果 |
|---|---|
| 集中药水 | 一场战斗限用 1 次，回合开始时使用，本场战斗集中 +2 |
| 黑暗药水 | 一场战斗限用 1 次，回合开始时使用，生成 1 个黑暗充能球；若携带暗核，初始储值为 18，否则为 14 |
| 释放药水 | 一场战斗限用 1 次，回合开始时使用，连续释放最左侧充能球 2 次；若没有充能球则无效果 |

### 状态规则

| 状态 | 规则 |
|---|---|
| 集中 | 影响闪电、冰霜、黑暗的数值 |
| 格挡 | 玩家格挡回合结束后清空 |
| 敌人护甲 | 敌人护甲不会自动清空 |
| 消耗 | 被消耗的牌本场战斗不再进入抽牌堆或弃牌堆 |
| 抽牌不足 | 抽牌堆不足时，立刻洗入弃牌堆继续抽 |

### 判定

| 维度 | 结论 |
|---|---|
| 难度目标 | 8/10 |
| 卡池规模 | 24 张，按 12 张构筑并叠加药水、遗物与充能球状态 |
| 核心机制 | 充能球槽、释放顺序、黑暗储值、冰霜绝缘破裂、集中、循环额外被动 |
| 三档回合 | 存在第 2、第 3、第 4 回合首次击杀线；第 4 回合线为低概率保底线 |
| 高风险高收益 | 最快线总成功率不足 50%，但上限最高；第 3 回合线更依赖集中与线缆稳定触发 |
| 偷跑控制 | 全量 3168 个构筑中无 100% 稳定线 |

## 资源审计

| 维度 | 结论 |
|---|---|
| 第一层资源选择 | 必须选对黑暗球、释放、集中/绝缘、药水和遗物资源包 |
| 错误遗物/慢速资源 | 数据磁盘低速保底线最高 4.9647%，不能靠牌序稳定通关 |
| 半正确资源 | 线缆集中线最高 36.8042%，低于暗核黑球主线且失败率超过 63% |
| 排序作用 | 球槽和释放顺序强烈影响概率，但没有正确球资源和药水/遗物时不能形成稳定路线 |
