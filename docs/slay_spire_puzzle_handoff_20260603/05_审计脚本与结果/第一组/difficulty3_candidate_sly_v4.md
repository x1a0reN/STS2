# 难度 3 候选：狡黠反击窗口

## 玩家题面

从 15 张候选牌中选择 8 张作为起始牌组，从 3 种药水中选择 1 种，并从 3 个遗物中选择 1 个带入战斗，在第 4 回合结束前击败敌人。

本题只按题面写明的效果结算。所有改动过的原版牌都用“（改）”标记。

### 初始状态

| 项目 | 数值 |
|---|---:|
| 玩家生命 | 22 |
| 每回合能量 | 3 |
| 每回合抽牌 | 5 |
| 起始牌组 | 从候选牌中选 8 张 |
| 药水 | 从药水池中选 1 种 |
| 遗物 | 从遗物池中选 1 个 |
| 敌人生命 | 78 |
| 回合上限 | 第 4 回合结束前击杀 |

### 本题关键词

| 关键词 | 规则 |
|---|---|
| 狡黠 | 你的回合中，带有“狡黠”的牌从手牌被弃掉时，立即免费打出；免费打出后进入弃牌堆 |
| 铃鸣 | 每回合最多主动打出 3 张牌；狡黠免费打出的牌不计入这 3 张 |
| 缠手 | 每回合第一次弃牌后，本回合不能再弃牌 |

### 敌人行动

| 回合 | 敌人行动 |
|---:|---|
| 1 | 攻击 8；本回合附加铃鸣、缠手；如果本回合没有任何狡黠牌被弃掉并免费打出，敌人获得 10 点护甲 |
| 2 | 攻击 14；本回合附加铃鸣、缠手；如果本回合没有任何狡黠牌被弃掉并免费打出，敌人获得 14 点护甲 |
| 3 | 攻击 22；本回合附加铃鸣、缠手 |
| 4 | 处决 30；本回合附加铃鸣、缠手 |

敌人护甲会保留，受到伤害时先扣护甲，再扣生命。

### 药水池

| 药水 | 效果 |
|---|---|
| 狡黠药水 | 一场战斗限用 1 次，0 费，本回合获得 1 点额外能量，然后弃掉 1 张手牌 |
| 破甲药水 | 一场战斗限用 1 次，0 费，给予敌人 2 层易伤 |
| 火焰药水 | 一场战斗限用 1 次，0 费，造成 18 点固定伤害；不受易伤加成；会先被敌人护甲吸收 |

### 遗物池

| 遗物 | 效果 |
|---|---|
| 锋利骰子 | 每回合第一次因弃牌免费打出背刺（改，狡黠）时，该牌伤害 +6 |
| 折返皮套 | 每回合第一次弃牌后，本回合获得 1 点额外能量 |
| 空心护符 | 每回合第一次弃牌后，获得 6 点格挡 |

### 状态规则

| 状态 | 规则 |
|---|---|
| 易伤 | 攻击牌伤害变为 1.5 倍，向下取整 |
| 虚弱 | 敌人本回合攻击伤害变为 0.75 倍，向下取整 |
| 格挡 | 玩家格挡回合结束后清空 |
| 敌人护甲 | 敌人护甲不会自动清空 |
| 抽牌不足 | 抽牌堆不足时，立刻洗入弃牌堆继续抽 |
