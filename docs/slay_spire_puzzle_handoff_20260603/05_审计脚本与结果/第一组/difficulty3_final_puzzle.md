# 难度 3 最终题目：双狡黠铃鸣与伪刃破绽

## 玩家题面

从 19 张候选牌中选择 8 张作为起始牌组，从 3 种药水中选择 1 种，并从 3 个遗物中选择 1 个带入战斗，在第 4 回合结束前击败敌人。

本题只按题面写明的效果结算。所有改动过的原版牌都用“（改）”标记。

### 选牌限制

| 限制 | 内容 |
|---|---|
| 二选一 | 预备（改）与生存者（改）必须且只能选择 1 张 |
| 总牌数 | 最终起始牌组为 8 张 |

### 初始状态

| 项目 | 数值 |
|---|---:|
| 玩家生命 | 26 |
| 每回合能量 | 3 |
| 每回合抽牌 | 5 |
| 起始牌组 | 按选牌限制从候选牌中选 8 张 |
| 药水 | 从药水池中选 1 种 |
| 遗物 | 从遗物池中选 1 个 |
| 敌人生命 | 80 |
| 回合上限 | 第 4 回合结束前击杀 |

### 本题关键词

| 关键词 | 规则 |
|---|---|
| 狡黠 | 你的回合中，带有“狡黠”的牌从手牌被弃掉时，立即免费打出；免费打出后进入弃牌堆 |
| 铃鸣 | 每回合最多主动打出 2 张牌；狡黠免费打出的牌不计入这 2 张 |
| 缠手 | 每回合第一次弃牌后，本回合不能再弃牌 |
| 伪刃 | 标有“非狡黠”的牌不会因狡黠免费打出；弃掉它们会触发误弃 |
| 识破 | 如果本回合主动打出背刺（改，狡黠）或影步（改，狡黠），每主动打出 1 张，敌人回合结束额外获得 8 点保留护甲 |
| 误弃 | 如果本回合弃掉的牌不是狡黠牌，敌人回合结束额外获得 6 点保留护甲 |
| 连环破绽 | 如果连续 3 个回合都至少有 1 张狡黠牌被弃掉并免费打出，第 4 回合开始时敌人失去 40 点生命；该失去生命不受护甲影响 |

### 敌人行动

敌人护甲会保留；受到伤害时先扣护甲，再扣生命。

| 回合 | 敌人行动 |
|---:|---|
| 1 | 攻击 8；本回合附加铃鸣、缠手、识破、误弃；如果本回合没有任何狡黠牌被弃掉并免费打出，敌人获得 14 点保留护甲 |
| 2 | 攻击 22；本回合附加铃鸣、缠手、识破、误弃；如果本回合没有任何狡黠牌被弃掉并免费打出，敌人获得 20 点保留护甲 |
| 3 | 攻击 22；本回合附加铃鸣、缠手、识破、误弃；如果本回合没有任何狡黠牌被弃掉并免费打出，敌人获得 8 点保留护甲 |
| 4 | 处决 30；本回合附加铃鸣、缠手、识破、误弃 |

### 判定

| 维度 | 结论 |
|---|---|
| 难度目标 | 3/10 |
| 卡池规模 | 19 张，符合难度 3 的 14-22 张范围 |
| 选择范围 | 8 张牌，只保留预备（改）/生存者（改）二选一约束；狡黠核心牌不再必选 |
| 核心机制 | 背刺负责狡黠输出，影步负责狡黠防御与连环破绽计数，终结吃全战斗狡黠累计 |
| 干扰结构 | 佯刺、匕首投掷、虚刃都像低费输出，但弃掉会触发误弃 |
| 三档回合 | 存在第 2、第 3、第 4 回合首次击杀线 |
| 偷跑控制 | 1530 个合法构筑中无 100% 稳定线；最高线仍属于双狡黠机制族 |

### 资源审计

| 维度 | 结论 |
|---|---|
| 第一层资源选择 | 玩家必须从开放候选池中识别双背刺、终结、影步、弃牌入口、药水和遗物共同构成的狡黠资源包 |
| 错误牌组 | 非狡黠牌只作为诱导资源；弃错会触发误弃并给敌人保留护甲 |
| 非主药水/遗物 | 火焰药水和锋利骰子可以形成快线变体，但最高仍低于狡黠药水+空心护符主线 |
| 排序作用 | 主动优先级和弃牌优先级决定狡黠触发窗口，但前提仍是先选对狡黠资源包 |
