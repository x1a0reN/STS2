# 第二组难度 6 最终：保留牌与预谋

## 玩家题面

> 作废：第二组题目不再作为有效产出或框架依据。

从 15 张候选牌中选择 6 张作为起始牌组，从 3 瓶候选药水中选择 1 瓶，并从 2 个候选遗物中选择 1 个，在第 5 回合结束前击败敌人。

本题采用固定自动结算顺序。每回合抽牌后，系统按“预谋 -> 过度预谋 -> 保留格挡 -> 触发斩 -> 预谋释放 -> 蓄势击 -> 稳击 -> 重斩 -> 打击 -> 防御 -> 空护”的顺序，依次打出手牌中能支付费用的牌；玩家只负责选择牌组、药水、遗物，不手动调整出牌顺序。

## 本题自定义规则

以下规则均为本题自定义机制，不是原版官方关键词；原版通用概念沿用《杀戮尖塔》官中称呼。

| 术语 | 规则 |
|---|---|
| 预谋 | 跨回合保留，最高 8 层 |
| 保留 | 保留牌若回合结束时仍在手牌中，不进入弃牌堆，留到下回合 |
| 预谋释放 | 按当前预谋造成额外伤害，然后清空预谋 |
| 过度预谋 | 获得预谋，但玩家失去生命，是低概率抢杀陷阱 |
| 过载反噬 | 回合结束时若仍有 7 点或以上预谋，玩家额外失去 4 点生命 |
