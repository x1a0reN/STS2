# 第二组难度 4 最终：伤口反噬与止血时机

## 玩家题面

> 作废：第二组题目不再作为有效产出或框架依据。

从 14 张候选牌中选择 7 张作为起始牌组，并从 3 瓶候选药水中选择 1 瓶，在第 5 回合结束前击败敌人。

本题采用固定自动结算顺序。每回合抽牌后，系统按“血价 -> 裂创 -> 伪血印 -> 缝合 -> 烙止 -> 转伤斩 -> 稳刺 -> 重斩 -> 打击 -> 防御”的顺序，依次打出手牌中能支付费用的牌；玩家只负责选择牌组和药水，不手动调整出牌顺序。

## 本题自定义规则

以下规则均为本题自定义机制，不是原版官方关键词；原版通用概念沿用《杀戮尖塔》官中称呼。

| 术语 | 规则 |
|---|---|
| 伤口 | 跨回合保留，最高 9 层 |
| 伤口反噬 | 回合结束时，玩家失去当前伤口层数 x2 的生命 |
| 止血时机 | 止血药水在本回合免除伤口反噬，并按当前伤口层数造成额外伤害 |
| 伤口转化 | 转伤斩按当前伤口层数造成额外伤害，然后清空伤口 |
| 洗牌 | 抽牌堆不足时，立即洗入弃牌堆继续抽 |

## 资源审计

| 维度 | 结论 |
|---|---|
| 第一层资源选择 | 必须同时处理伤口来源、转伤兑现、伤口反噬生存窗口 |
| 忽略止血/伤口转化 | 护壳药水最高 15.5000%，狂血药水最高 13.2381%，均低于 D4 误选上限 |
| 伪血印陷阱 | 伪血印最高 11.4286%，会削掉转伤层数，不能稳定逃课 |
| 排序作用 | 固定顺序只决定伤口何时被清空；缺少伤口兑现或生存窗口时成功率会塌陷 |
