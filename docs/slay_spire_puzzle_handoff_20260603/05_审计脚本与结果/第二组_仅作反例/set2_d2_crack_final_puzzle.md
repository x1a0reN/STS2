# 第二组难度 2 最终：破甲标记与伪破绽

## 玩家题面

> 作废：第二组题目不再作为有效产出或框架依据。

从 15 张候选牌中选择 7 张作为起始牌组，并从 3 瓶候选药水中选择 1 瓶，在第 4 回合结束前击败敌人。

本题采用固定自动结算顺序。每回合抽牌后，系统按“审视 -> 刻痕斩 -> 伪破绽 -> 破甲收束 -> 穿刺 -> 护身刺击 -> 重斩 -> 打击 -> 防御”的顺序，依次打出手牌中能支付费用的牌；玩家只负责选择牌组和药水，不手动调整出牌顺序。

## 本题自定义规则

以下规则均为本题自定义机制，不是原版官方关键词；原版通用概念沿用《杀戮尖塔》官中称呼。

| 术语 | 规则 |
|---|---|
| 破甲 | 跨回合保留，最高 8 层 |
| 破甲收束 | 根据当前破甲层数造成额外伤害，然后清空破甲 |

| 洗牌 | 抽牌堆不足时，立即洗入弃牌堆继续抽 |
