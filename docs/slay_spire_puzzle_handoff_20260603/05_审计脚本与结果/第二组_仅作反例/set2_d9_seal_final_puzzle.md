# 第二组难度 9 最终：封印牌与解封阈值

## 玩家题面

> 作废：第二组题目不再作为有效产出或框架依据。

从 15 张候选牌中选择 7 张作为起始牌组，从 3 瓶候选药水中选择 1 瓶，并从 2 个候选遗物中选择 1 个，在第 5 回合结束前击败敌人。

本题采用固定自动结算顺序。每回合抽牌后，系统按“解封钥匙 -> 深封 -> 护钥 -> 延锁 -> 封印击 -> 解封释放 -> 稳击 -> 重斩 -> 打击 -> 防御 -> 空牌”的顺序，依次打出手牌中能支付费用的牌；玩家只负责选择牌组、药水、遗物，不手动调整出牌顺序。

## 本题自定义规则

以下规则均为本题自定义机制，不是原版官方关键词；原版通用概念沿用《杀戮尖塔》官中称呼。

| 术语 | 规则 |
|---|---|
| 解封钥匙 | 跨回合保留，达到 3 点后视为已解封 |
| 封印层数 | 跨回合保留，最高 7 点 |
| 未解封 | 封印击与解封释放只能造成低伤害，并增加封印层数 |
| 已解封 | 封印击变为高伤害；解封释放按封印层数造成额外伤害 |
| 封印反噬 | 回合结束时，超过 2 点的封印层数会造成生命损失 |
