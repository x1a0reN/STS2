# 输出模板

本文件用于规范最终题面、审计摘要、难度报告和交接文档的输出。

## 原则

根据用户需求选择输出粒度。

如果用户说“只要最终题目”，不要输出推导过程。

如果用户说“复核”“审计”“确认是否对”，必须输出审计结论和关键证据。

如果用户说“分享给别人”“做成框架”，输出可独立阅读的框架文档或 skill。

## 用户要求与输出选择

### 用户只要题目

输出：

- 最终题面
- 代表解法族

不输出：

- 推导过程
- 审计细节
- 修改理由

### 用户要复核

输出：

- 审计结论
- 关键越界路线
- 是否通过
- 是否需要改

### 用户要框架

输出：

- 方法论
- 难度标准
- 审计流程
- 判断模板

### 用户要继续改题

输出：

- 改后的题面
- 必要审计摘要
- 是否解决原问题

## 术语统一

推荐术语：

- 代表解法族
- 快速高风险
- 标准中速
- 慢速稳健
- 同族变体
- 覆盖路线
- 越界偷跑
- 100% 稳定线
- 概率梯度

避免术语：

- 唯一答案
- 标准答案
- 真解

除非题目确实要求唯一解。
