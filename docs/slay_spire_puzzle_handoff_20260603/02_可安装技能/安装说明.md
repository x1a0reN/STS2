# 技能安装说明

## 一键复制

在 PowerShell 中进入本目录后执行：

```powershell
.\install_skills.ps1
```

默认会复制到：

```text
$env:USERPROFILE/.codex/skills
```

如果承接者使用自定义 Codex skills 目录：

```powershell
.\install_skills.ps1 -CodexSkillsDir "D:/path/to/.codex/skills"
```

## 必须确认

安装后至少确认以下两个 skill 存在：

- `slay-spire-mod-design-director`
- `slay-spire-puzzle-control-skill`

其中 `slay-spire-puzzle-control-skill` 是本轮专门抽象出的出题可控性框架。后续审题、改题、做概率控制时必须优先读取。
