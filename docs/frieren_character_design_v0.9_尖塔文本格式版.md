# 芙莉莲独立角色设计稿 v0.9 尖塔文本格式版

目标：按《杀戮尖塔》卡面语法重写文本，不改变 v0.5 的机制和数值。  
原则：基础效果先写；条件收益用前缀；关键词独立成句。

## 1. 关键词

| 关键词 | 描述 |
|---|---|
| 法术 | 芙莉莲多数专属牌标签。打击、防御不算。 |
| 解析 X | 法术造成未格挡伤害时，消耗1层，使敌人失去2点生命。 |
| 看破 X | 下一张消耗解析的法术，使每层解析改为失去3点生命。 |
| 隐匿魔力 | 达到10层时，失去10层，获得1层解放。上限19层。 |
| 解放 X | 下一张法术额外打出1次。消耗1层解放。 |
| 抑制 X | 回合结束时，如果本回合没有打出攻击，获得X层隐匿魔力。 |
| 咏唱 X | 延迟X回合后结算。费用和目标在打出时确定。 |
| 回路 | 每手动打出3张法术，触发1次回路。 |
| 普通魔法 | 小型功能魔法标签，用于回忆和普通魔法流。 |
| 回忆 | 每场战斗首次手动打出不同名普通魔法时，获得1层回忆。 |

补充规则：

1. 解放打出的复制法术不再触发解放。
2. 咏唱结算不推进回路。
3. 每3层回忆触发一次旅途奖励：抽1张牌、获得7点格挡、获得3层隐匿魔力，循环。
4. 生成的普通魔法每回合最多触发1次回忆。
5. 卡面上的 `解放：` 表示该牌因解放被额外打出时，追加结算该效果。

## 10. 文本规范

本版不新增缩写词，不改变卡名、数量、费用、稀有度和数值。  
升级栏写完整升级后文本，不写“+2伤害”这类差值。  
固定写法：伤害写“点伤害”，格挡写“点格挡”，层数写“层”，抽牌写“张牌”。  
条件收益用前缀：`解放：`、`看破：`、`回忆：`、`击杀：`、`目标有解析：`、`目标8层解析：`。  
关键词单独成句：`固有。`、`保留。`、`消耗。`
